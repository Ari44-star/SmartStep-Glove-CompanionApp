# 🧤 SmartStep: Asistencia Sensorial Inteligente

Es un sistema de asistencia sensorial que combina hardware embebido (Arduino) y una Companion App móvil bajo el patrón MVVM, diseñado para mejorar la movilidad de personas con discapacidad visual.

## 🚀 Características
* Detección de Obstáculos: Sensor ultrasónico HC-SR04 (< 10 cm para alerta crítica).
* Alertas Hápticas: Motor vibrador con intensidad regulada por PWM.
* Interfaz Accesible: Cumplimiento de normas **WCAG 2.1** (alto contraste y botones masivos).
